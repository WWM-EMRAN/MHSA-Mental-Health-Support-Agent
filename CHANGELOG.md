# Changelog

All notable changes to the MHSA (Mental Health Support Agent) project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-23

### Added
- Initial release of MHSA Mental Health Support Agent
- Core conversational agent with GPT-4 integration
- Three-tier crisis detection system (critical, high, medium levels)
- Automatic crisis resource provision
- Conversation management and persistence
- SQLite database with SQLAlchemy ORM
- User, Conversation, Message, and SessionManager models
- Sentiment analysis for conversation tracking
- CLI application interface with colorized output
- Comprehensive system prompts for empathetic responses
- Crisis resources (US and international hotlines)
- Evidence-based coping strategies (anxiety, depression, stress)
- Sleep hygiene and lifestyle recommendations
- Complete test suite (20 unit tests)
- Extensive documentation:
  - README with quick start guide
  - Architecture documentation
  - Complete API reference
  - Crisis protocol documentation
  - Development guide
  - Contributing guidelines
- Configuration management via .env files
- Logging system with file and console output
- MIT License with mental health disclaimers
- Demo script for functionality showcase

### Security
- Safety-first design with crisis detection
- No storage of sensitive medical information
- Environment-based API key management
- Proper data privacy considerations

### Documentation
- Comprehensive README with badges and quick start
- Architecture overview with diagrams
- Complete API documentation with examples
- Crisis detection and response protocols
- Development setup and contribution guidelines
- Code of conduct for contributors

### Testing
- Crisis detection tests (8 tests)
- Database operation tests (6 tests)
- Sentiment analysis tests (6 tests)
- 100% test passing rate

## Notes

This is a capstone project for the Google 5-day Intensive Agentic AI course. 

**Important Disclaimer**: This software is provided as a mental health support tool and is NOT intended to replace professional mental health services, therapy, counseling, or medical treatment.

For crisis support:
- US: Call or text 988 (Suicide & Crisis Lifeline)
- Emergency: Call 911 or local emergency services
